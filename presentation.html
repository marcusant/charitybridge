🚀 Welcome to Charity Bridge

Bridging the gap between nonprofit needs and technical experts

Charity Bridge

An elegant dual-portal ecosystem designed to coordinate, optimize, and streamline volunteer matching for nonprofit social groups.

Powered by immediate local AI request classification, WCAG-compliant accessible design, and real-time synchronized data structures.

🏢 For Nonprofits

Describe challenges, receive instant first-aid response guidelines, and connect with technical specialists.

🧑‍💻 For Tech Volunteers

Browse active tickets matching your professional profile, view AI problem diagnosis, and claim tasks instantly.

⚠️ The Non-Profit Bottleneck

Why traditional volunteer coordination fails

Nonprofit organizations (NGOs) are frequently targeted by cyberthreats, suffer from obsolete IT infrastructures, and face data fragmentation, yet they lack the resources to hire dedicated technicians.

🚨 Critical Incidents

Hacked accounts, system failures, and compliance gaps require immediate expert triage—not slow, generic email exchanges.

🔍 Skills Mismatch

Generic volunteer boards fail to filter by precise, granular requirements (e.g. DNS routing, grant auditing, QuickBooks setups).

⏳ Lost Time

NGOs spend days trying to formulate tech requests, while volunteers spend hours looking for tasks that match their schedule and profile.

🚧 Accessibility Barriers

Obsolete systems and portals exclude users relying on keyboard-only focus or screen readers, missing out on valuable talent.

✅ The Dual-Portal Solution

A synchronized, fast, and structured pipeline

Charity Bridge resolves these bottlenecks by splitting the workflow into two dedicated, beautifully synchronized environments.

🔑 1. The NGO Support Portal

Simplifies request filing. Features a clean form validated in real time.

  • Local AI dynamic keyword category categorization.
  • Instant, actionable response plan to act as immediate first-aid.
  • Automatic creation of a structured support ticket.

🔑 2. The Volunteer Portal

A statistics-driven dashboard optimized for rapid triage.

  • Dynamic category pills and real-time search filtration.
  • Deep-dive detail panel rendering AI diagnostic summaries.
  • One-click task claiming with immediate status sync.
💾

Synchronized via Local Storage

All created tickets, claim status shifts, and volunteering hour meters are updated and synchronized globally across portals on the fly.

⚙️ Technical & Quality Standards

Built with zero external weight, fully responsive, and highly accessible

🎨 Unified Visual Tokens

Single Source of Truth: Every layout block, card shape, font style, and transition curve is loaded from a single central stylesheet.

Designed using a semantic `:root` HSL color scale, perfectly adaptable for dark mode scaling.

♿ WCAG 2.1 AA A11y

100% Keyboard Navigable: Replaced all inaccessible hidden radio selections with an off-screen `.sr-only` setup, adding prominent `:focus-visible` rings.

All graphical icons and decorative elements are wrapped with precise aria-labels for screen readers.

🧠 Local AI Categorizer

Semantic Keyword Parsing: Features a pure JavaScript regex keyword cluster classifier.

Scans the problem description on-the-fly to predict issues in Cybersecurity, IT Infrastructure, Finance, Software, or Comms automatically.

🛡️ Robust Form Validations

Dynamic Inline Feedback: Legacy blocking popups are replaced with descriptive inline warning banners.

Event triggers clean error flags instantly upon typing, focusing on the first failing element upon submission.
